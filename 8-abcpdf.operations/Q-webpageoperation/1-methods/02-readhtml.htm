ReadHtml Function    

Read a string of HTML into a paged PDF document.

 

    

Syntax		  

[C#]
ReadHtml(string html)

[Visual Basic]
ReadHtml(html As String)

 

    

Params		  
Name Description
html The HTML to read.

 

    

Notes		  

Read a string of HTML into a paged PDF document.

On return all Doc properties are reset to default values.
