Type Default Value Read Only Description
[C#]
GraphicsState

[Visual Basic]
GraphicsState		 null Yes The current graphics state.

 

    

Notes		  

The current graphics state. This broadly corresponds with the 'Graphics State' as described in the PDF Specification.

The current graphics state is what is saved and restored using the PDF save and restore operators.
