ContentStreamScanner Function    

Create a new ContentStreamScanner.

 

    

Syntax		  

[C#]
ContentStreamScanner()
ContentStreamScanner(ContentStreamScanner sibling)

 

    

Params		  
Name Description
sibling A scanner which contains cached data.

 

    

Notes		  

Create a new ContentStreamScanner.

A ContentStreamScanner will cache useful resources as they are needed. If you are creating multiple ContentStreamScanners for the same document you can share this cache to increase efficiency.

To share cached data in this way pass in a sibling scanner previously used to scan a content stream in the same doc.

 

    

Example		  

None.
