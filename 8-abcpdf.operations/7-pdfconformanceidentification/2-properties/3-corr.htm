Corr Property    

 

Type Default Read Only Description
[C#]
string

[Visual Basic]
String		 null No The conformance corrigendum identifier.

 

    

Notes		  

This property indicates the corrigendum to a part of the conformance specification. For PDF/A, the value is the corrigendum number and year, separated by a colon. There is currently no corrigendum to PDF/A-2.

 

    

Example		  

None.
