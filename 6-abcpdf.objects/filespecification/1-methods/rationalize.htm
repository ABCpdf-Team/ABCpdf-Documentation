Rationalize Function    

Removes any obsolescent and redundant entries.

 

    

Syntax		  

[C#]
void Rationalize()

 

    

Params		  
Name Description
none n/a

 

    

Notes		  

Removes any obsolescent and redundant entries.

This will converts the object to be compliant with the changes recommended in the PDF 1.7 standard.
