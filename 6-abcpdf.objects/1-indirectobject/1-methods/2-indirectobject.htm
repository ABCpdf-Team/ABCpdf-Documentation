IndirectObject Constructor    

IndirectObject Constructor.

 

    

Syntax		  

[C#]
IndirectObject()

 

    

Params		  
Name Description
none  

 

    

Notes		  

Creates an indirect object containing a NullAtom.

Typically you will need to replace the NullAtom before you can do anything useful with the object.

 

    

Example		  

None.
