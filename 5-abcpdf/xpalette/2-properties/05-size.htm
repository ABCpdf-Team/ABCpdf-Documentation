Type Default Value Read Only Description

[Visual Basic]
Integer
 256 No The target number of colors in the palette.

 

    

Notes		  

The target number of colors in the palette.

The valid range for this property is between 2 and 256.

The target is a maximum. The actual number of colors that will appear in the palette will depend on the Definition and also perhaps on the content of the image.
