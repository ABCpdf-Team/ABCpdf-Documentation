Clear Function    

Clears the document.

 

    

Syntax		  

[C#]
void Clear()

[Visual Basic]
Sub Clear()

 

    

Params		  
Name Description
none  

 

    

Notes		  

Use this method to release resources and return the document to a just-created state.
