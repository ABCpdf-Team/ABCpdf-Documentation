# Examples

There are many examples throughout the documentation for common tasks such as:
- Adding styled text: see 5-abcpdf/doc/1-methods/addtextstyled.htm
- Flowing text: see 4-examples/02-textflow.htm
- Adding images: see 4-examples/04-image.htm

Every major method/property includes a code sample. For example:
- `AddText`: see 5-abcpdf/doc/1-methods/addtext.htm
- `AddImageFile`: see 5-abcpdf/doc/1-methods/addimagefile.htm

Operations examples include:
- Rendering HTML pages: `AddImageUrl`
- Paged HTML renders: `AddImageToChain`
- Watermarking: `AddImageCopy`
- Appending PDFs: `Append`
- Drawing pages from one PDF into another: `AddImageDoc`

For in-situ examples, refer to the example website installed with ABCpdf.
