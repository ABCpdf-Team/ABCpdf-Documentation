# XPoint Constructor

XPoint Constructor.

## Syntax

```csharp
<a href="../default.htm">XPoint</a>()
<a href="../default.htm">XPoint</a>(string value)<br> <a href="../default.htm">XPoint</a>(double x, double y)
```

## Params

| Name | Description |
| --- | --- |
| value | A string defining the initial point in the format "x y". |
| x | The x coordinate for the point. |
| y | The y coordinate for the point. |

## Notes

These methods construct an XPoint object.
