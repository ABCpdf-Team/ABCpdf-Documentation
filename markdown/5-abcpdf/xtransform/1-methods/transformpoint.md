# TransformPoint  Function

Applies this transform to a specified point.

## Syntax

```csharp
void TransformPoint(Point[] point)<br> void TransformPoint(PointF[] point)<br> void TransformPoint(<a href="../../xpoint/default.htm">XPoint</a> point)<br> void TransformPoint(ref double x, ref double y)
```

## Params

| Name | Description |
| --- | --- |
| point | The point to be transformed. |
| x | The x coordinate of a point to be transformed. |
| y | The y coordinate of a point to be transformed. |
