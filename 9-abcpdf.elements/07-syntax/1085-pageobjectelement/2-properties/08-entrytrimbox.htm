EntryTrimBox Property    

 

Type Default Value Read Only Description
[C#]
RectangleElement

[Visual Basic]
RectangleElement		 null No Represents the "TrimBox" entry of the page object object.

 

    

Notes		  

Represents the "TrimBox" entry of the page object object.

It is an optional entry defined as part of the PDF 1.3 specification.

It contains a RectangleElement.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 30, page 77.

Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 3.27, page 23.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 31, page 104.
