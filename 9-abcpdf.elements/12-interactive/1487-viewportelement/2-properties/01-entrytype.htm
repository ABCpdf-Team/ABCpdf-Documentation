EntryType Property    

 

Type Default Value Read Only Description
[C#]
string		 null No Represents the "Type" entry of the viewport dictionary object.

 

    

Notes		  

Represents the "Type" entry of the viewport dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a string representing a PDF name object.

If provided, this item must always take the value "Viewport".

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 260, page 479.

Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 8.109, page 48.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 265, page 592.
