GetEnumerator Function    

Returns an enumerator that iterates through the collection.

 

    

Syntax		  

[C#]
virtual IEnumerator<T> GetEnumerator()

 

    

Params		  
Name Description
return An enumerator for the collection.

 

    

Notes		  

Returns an enumerator that iterates through the collection.

Gets a value enumerator for the Collection.

 

    

Example		  
None.
