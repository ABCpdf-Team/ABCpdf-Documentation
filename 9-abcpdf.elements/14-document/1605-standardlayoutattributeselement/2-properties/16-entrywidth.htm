Notes		  

Represents the "Width" entry of the standard layout attributes object.

It is an optional entry defined as part of the PDF 1.0 specification.

This property may contain one of two different types:.

1) A double representing a PDF numeric object.

2) A string representing a PDF name object.

The PDF specification states that this item assumes a value of "Auto" if no value has been provided.

If provided, this item must always take the value "Auto".

For definitive details see:.
