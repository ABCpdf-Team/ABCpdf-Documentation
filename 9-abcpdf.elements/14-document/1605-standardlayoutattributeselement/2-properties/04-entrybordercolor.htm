EntryBorderColor Property    

 

Type Default Value Read Only Description
IList<double?> null No Represents the "BorderColor" entry of the standard layout attributes object.

 

    

Notes		  

Represents the "BorderColor" entry of the standard layout attributes object.

It is an optional entry defined as part of the PDF 1.5 specification.

It contains an array which contains doubles, representing PDF numeric objects.

Items in this array should have a minimum value of 0 and a maximum of 1.
